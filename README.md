## 設定環境

* 安裝composer
* 安裝node.js
* 安裝php

## 安裝package

pull下來後 下下列指立

server `composer require`

client `npm install`

需複製`.env.example`為`.env`

## 啟動指令

server 

`php artisan serve`

client

`develop:npm run watch `

`production:npm run production`

最後下

`php artisan key:generate`
